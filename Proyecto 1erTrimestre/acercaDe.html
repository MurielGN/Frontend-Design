HTML


Uso modular de css externos

He utilizado la referencia a css externos múltiple para poder reutilizar el código.


Uso de hsla

Juego con los hsla para dar color rojo como principal y azul como el traje para detalles. No uso colores complementarios si no que juego con la saturación luminosidad y transparencia.

Grid


Uso anidado de grids

He usado el display grid de forma anidada con el objetivo de hacer más sencilla la maquetación. He usado grids genéricos para colocar un main y un aside. Luego dentro de he vuelto a usar otro grid para ordenar los elementos más pequeños de su interior. Así sucesivamente.


Anidando con FlexBox

En algunos aspectos de la página como los iconos del head, he utilizado un FlexBox dentro de un grid. De esta forma he puedo hacer los iconos algo responsive acumulando en columnas si es necesario.

SASS


Uso de @mixin

Por la propia estructura de página de formación, he utilizado dos @mixin con sus @include. Con esto he ahorrado repetir muchas líneas de código, además de conseguir un código más limpio


Uso variables e @imports

Utilizo variables para los colores y imports al inicio de cada .scss para tener los colores disponibles


Uso de anidación y especificidad

Utilizo la facilidad de anidación de SASS para simplificar mi .scss y asegurar la especificidad.

Investigación


Iconos con fondo transparente

Para poder tener en el head iconos con fondo transparente he necesitado dos investigaciones. La primera hay que dejar el fondo vacío con photoshop. Aun con esto el navegador deja el fondo de la imagen en blanco por defecto lo que se arregla con la propiedad background-color: transparent


Aumentar espacio entre elementos de una lista

Por motivos estéticos de la página principal necesitaba aumentar el espacio entre elementos de una lista. Se puede conseguir con la propiedad de las listas line-height
